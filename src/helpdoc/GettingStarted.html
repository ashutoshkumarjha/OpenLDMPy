Getting Started

This software is a pure-Python application, supported on Windows, Linux and macOS.

It runs as a standalone desktop application (PyQt5 GUI) and also works headlessly from the command line for batch processing or HPC use — both through the same entry point, OpenLDM.py. There is no R or rpy2 runtime dependency: the modeling pipeline (data I/O, transition matrices, suitability modeling, allocation, accuracy assessment, cartographic export) is implemented natively in Python using

Geospatial I/O - rasterio, GDAL, geopandas

Modeling - scikit-learn (logistic regression, random forest, SVM, neural network)

Parallel processing - joblib

Cartographic export - matplotlib

GUI - PyQt5

A QGIS Processing plugin is planned as a separate, future migration subtask (not yet built on top of the core package). This software has been tested with the following software versions:

Python 3.10

PyQt5 5.15

(OpenLDM) v1.0 IIRS
