Installing the standalone program

Getting Started ››

There is no separate installer program — OpenLDM is installed by cloning the repository and creating the Conda/Mamba environment described in System requirements:

git clone https://github.com/ashutoshkumarjha/OpenLDMPy.git

cd OpenLDMPy

mamba env create -f environment.yml

mamba activate openldm

This works the same way on Windows, Linux and macOS. Once the environment is created and activated, continue to Starting Program.
